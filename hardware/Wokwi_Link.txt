Office Monitoring System
Hardware Simulation (Representative Room)

This Wokwi project demonstrates the representative hardware circuit for one office room using an ESP32 development board.

The simulation includes:
- ESP32-WROOM-32 Development Board
- 2 Yellow LEDs representing Fans
- 3 Blue LEDs representing Lights
- 220Ω current-limiting resistors for each output

GPIO Mapping:
- GPIO13 → Fan 1
- GPIO12 → Fan 2
- GPIO14 → Light 1
- GPIO27 → Light 2
- GPIO26 → Light 3

This representative circuit can be replicated for the remaining rooms in the office, satisfying the hardware simulation requirement of the project.
